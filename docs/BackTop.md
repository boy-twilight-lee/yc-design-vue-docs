### BackTop Props

| 参数名 | 描述 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| `visibleHeight` | 滚动高度达到此值时显示回到顶部按钮 (px) | `number` | `200` |
| `targetContainer` | 设置需要监听其滚动事件的元素 | `TargetContainer` | `window` |
| `easeing` | 滚动的缓动函数 | `string` | `'quartOut'` |
| `duration` | 滚动动画的持续时间 (ms) | `number` | `500` |

### BackTop Slots

| 插槽名 | 描述 |
| :--- | :--- |
| `default` | 自定义回到顶部按钮的内容 |
