### 单向

通过设置 one-way ，使用单向模式的穿梭框。

<div class="cell-demo vp-raw">
  <yc-transfer
    :data="data"
    :default-value="value"
    one-way />
</div>

<script setup>
const data = Array(8)
  .fill(undefined)
  .map((_, index) => ({
    value: `option${index + 1}`,
    label: `Option ${index + 1}`,
  }));
const value = ['option1', 'option3', 'option5'];
</script>
