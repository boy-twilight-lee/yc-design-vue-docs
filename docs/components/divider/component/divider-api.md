## API

### Props

| 参数        | 说明           | 类型          | 默认值         |
| ----------- | -------------- | ------------- | -------------- |
| direction   | 分割线方向     | `Direction`   | `'horizontal'` |
| orientation | 分割线文字位置 | `Orientation` | `'center'`     |
| type        | 分割线类型     | `DividerType` | `'solid'`      |
| size        | 分割线粗细     | `number`      | `1`            |
| margin      | 分割线边距     | `number`      | -              |

### Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 分割线文字内容 |
