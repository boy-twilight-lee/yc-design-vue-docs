### Drawer全局方法

<span style="font-size:16px;">`Drawer` 提供的全局方法，可以通过以下三种方法使用：</span>

1. 通过 `this.$drawer` 调用
2. 在 Composition API 中，通过 `getCurrentInstance().appContext.config.globalProperties.$drawer` 调用
3. 导入 Drawer，通过 `Drawer` 本身调用
