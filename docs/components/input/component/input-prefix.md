### 前缀与后缀

通过指定 <yc-tag>prefix</yc-tag> 和 <yc-tag>suffix</yc-tag> 插槽来在输入框内添加前缀和后缀。

<div class="cell-demo vp-raw">
  <yc-space
    direction="vertical"
    size="large">
    <yc-input
      :style="{ width: '320px' }"
      placeholder="Please enter something"
      allow-clear>
      <template #prefix>
        <icon-user />
      </template>
    </yc-input>
    <yc-input
      :style="{ width: '320px' }"
      placeholder="Please enter something"
      allow-clear>
      <template #suffix>
        <icon-info-circle />
      </template>
    </yc-input>
  </yc-space>
</div>
