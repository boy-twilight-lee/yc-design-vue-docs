### 基础用法

按钮分为 <yc-tag>primary</yc-tag> - 主要按钮、<yc-tag>secondary</yc-tag> - 次要按钮（默认）、<yc-tag>dashed</yc-tag>- 虚线按钮、<yc-tag>outline</yc-tag> - 线形按钮、<yc-tag>text</yc-tag> - 文本按钮五种类型。
