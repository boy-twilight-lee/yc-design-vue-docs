### 显示输入框

当设置 <yc-tag>show-input</yc-tag> 时，将显示输入框。

<div class="cell-demo vp-raw">
  <yc-space
    direction="vertical"
    size="large">
    <yc-slider
      :default-value="10"
      :style="{ width: '300px' }"
      show-input />
    <yc-slider
      :default-value="[10, 20]"
      :style="{ width: '380px' }"
      range
      show-input />
  </yc-space>
</div>
