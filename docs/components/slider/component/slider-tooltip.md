### 自定义提示

通过设置 <yc-tag>format-tooltip</yc-tag> 可以自定义提示文字。

<div class="cell-demo vp-raw">
  <yc-slider
    :min="0"
    :max="50"
    :style="{ width: '200px' }"
    :format-tooltip="formatter" />
</div>

<script setup>
const formatter = (value) => {
  return `${Math.round((value / 50) * 100)}%`;
};
</script>
