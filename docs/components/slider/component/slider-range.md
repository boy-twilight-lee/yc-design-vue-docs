### 范围选择

通过设置 `range` 可开启范围选择，此时 `modelValue` 为数组。

<div class="cell-demo vp-raw">
  <a-slider
    v-model="value"
    :style="{ width: '300px' }"
    range />
</div>

<script setup>
import { ref } from 'vue';
const value = ref([5, 10]);
</script>
