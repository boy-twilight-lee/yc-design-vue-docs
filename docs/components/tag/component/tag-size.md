### 标签的尺寸

标签的大小分为：`small`、`medium`、`large` 三种，可以在不同场景下选择合适按钮尺寸。推荐及默认尺寸为 `medium`。

<div class="cell-demo vp-raw">
  <yc-space>
    <yc-tag size="large">Large</yc-tag>
    <yc-tag>Medium</yc-tag>
    <yc-tag size="small">Small</yc-tag>
  </yc-space>
</div>
