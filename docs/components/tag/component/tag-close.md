### 可关闭标签

通过 <yc-tag>closable</yc-tag> 属性控制标签是否可关闭。可关闭标签可通过 <yc-tag>close</yc-tag> 事件执行一些关闭后操作，也可通过 <yc-tag>visible</yc-tag> 属性控制标签的显示或隐藏。

    <yc-tag closable>Tag</yc-tag>
    <yc-tag closable>
      <template #icon>
        <icon-star />
      </template>
      Tag
    </yc-tag>
  </yc-space>
</div>
