### FAQ

`<MenuItem> `和 `<SubMenu> `组件的 `path` 属性为必填
<br />
在 `<Menu>` 组件中使用 `<MenuItem>` 和 `<SubMenu>` 组件时，请传入唯一的 path 属性。
<br />
组件内部在进行计算时会依赖此值，如果没有赋值会导致部分场景下异常
