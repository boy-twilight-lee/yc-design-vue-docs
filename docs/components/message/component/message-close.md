### 可关闭

设置 `closable` 来显示关闭按钮。

<div class="cell-demo vp-rwa">
  <yc-button
    @click="
      this.$message.info({
        content: 'This is an info message!',
        closable: true,
      })
    "
    >Closeable Message</yc-button
  >
</div>
